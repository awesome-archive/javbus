# javbus
A spider works on javbus.in

## Usage

```shell
usage: spider.py [-h] [-i INPUT] [-o OUTPUT] [-p PROCESS]

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Specifies the last results file
  -o OUTPUT, --output OUTPUT
                        Specifies the output file name, DEFAULT javbus.json
  -p PROCESS, --process PROCESS
                        Number of Subprocess, DEFAULT cpu_kernel_counts
```
